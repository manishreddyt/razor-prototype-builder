## 📋 Quick Demo Flow (2 minutes)

### Step 1: Navigate to Agents
```
1. Open http://localhost:8081/agents
2. Scroll to "Social Commerce" card (Instagram agent)
3. Notice the "Enable" button (not "Deploy")
```

### Step 2: Optional - Configure Goal
```
1. Click "Configure" button
2. Select template: "Auto-respond to product inquiries"
3. AI generates goal
4. Click "Save"
5. Status changes to "Configured"
```

### Step 3: Open Setup Wizard
```
1. Click "Enable" button
2. Setup wizard opens with 4-step progress bar
```

### Step 4: Connect Instagram Account
```
1. See Instagram branding (purple-to-pink gradient)
2. Read requirements:
   • Instagram Business or Creator account
   • Connected to Facebook Page
   • Admin access
3. Click "Connect Instagram Account"
4. Wait 2 seconds (mock OAuth loading)
5. Account connects: @fashionstore.in (12.5K followers)
6. See "What's Next?" section
7. Click "Next"
```

### Step 5: Configure Auto-Replies
```
1. Review greeting message:
   "Hi! 👋 Thanks for reaching out. How can I help you today?"
2. Optionally edit the greeting
3. See character counter (x/500)
4. Review 3 quick reply previews:
   • "What are your sizes?" → Size chart
   • "How much does shipping cost?" → FREE shipping info
   • "What is your return policy?" → 7-day returns
5. Click "Next"
```

### Step 6: Set Automation Rules
```
1. See 3 automation toggles (all ON by default):

   ✅ Auto-Reply to DMs (Recommended)
      "Respond to inquiries within 2 minutes"
      Example: "Do you have this in M?" → Availability + link

   ✅ Convert Comments to Sales (High Impact)
      "Turn comments into DM conversations"
      Example: "Want this!" → Product card + payment link

   ✅ Cart Abandonment Reminders
      "Remind customers within 24 hours"
      Example: Viewed yesterday → Reminder with link

2. Optionally toggle any automation off
3. Optionally enter product catalog URL
4. Click "Next"
```

### Step 7: Review & Enable
```
1. See configuration summary:

   Connected Account:
   • @fashionstore.in
   • 12.5K followers
   • Profile picture

   Enabled Automations:
   • ✓ Auto-reply to DMs
   • ✓ Convert comments to sales
   • ✓ Cart abandonment reminders

   Greeting Message:
   • "Hi! 👋 Thanks for reaching out..."

   What Happens Next:
   • Agent monitors DMs 24/7
   • Automated responses go live immediately
   • View conversations in dashboard
   • Pause or modify settings anytime

2. Click "Enable Instagram Agent" (green button)
```

### Step 8: Success!
```
1. Wizard closes
2. Agent status changes to "Deployed"
3. Button changes to "Pause"
4. Agent is now active!
```

## 🐛 Common Issues

### Wizard doesn't open
**Check**: Agent status should be "draft" or "configured"
**Fix**: If "deployed", click "Pause" first, then "Enable" again

### Can't click "Next"
**Check**: Step requirements met?
- Step 1: Account must be connected
- Step 2: Greeting message not empty
**Fix**: Complete required fields

### Changes not saving
**Issue**: Closing wizard mid-flow
**Fix**: Complete all 4 steps and click "Enable Instagram Agent"
